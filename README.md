# ResearchGate.in

![ResearchGate.in Logo](assets/images/logo.svg)

ResearchGate.in is an AI-powered research assistant that helps users with academic and research queries through a modern chat interface. Users can ask questions about scientific topics, historical events, literature reviews, methodologies, or any other research area.

## 🌟 Features

- **AI-Powered Research Assistant**: Get instant, accurate responses to your research questions
- **User-Friendly Interface**: Clean, intuitive chat interface for seamless interaction
- **No Login Required**: Quick access without registration
- **Mobile Responsive**: Works perfectly on all devices
- **Markdown Support**: Format your queries with basic markdown syntax
- **Local Chat History**: Chat history saved locally for your convenience

## 🚀 Getting Started

### Visit the Website

Simply go to [ResearchGate.in](https://researchgate.in) to start using the research assistant.

### Running Locally

If you want to run the application locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/researchgatein.git
   ```

2. Navigate to the project directory:
   ```bash
   cd researchgatein
   ```

3. Open `index.html` in your web browser.

## 📝 Usage Limits

To ensure fair usage and maintain service quality, ResearchGate.in has the following limitations:

- Non-registered users: 10 queries per hour
- (Future feature) Registered users: 50 queries per hour
